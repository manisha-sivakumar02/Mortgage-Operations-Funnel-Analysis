# Mortgage Application Funnel Analysis & Process Optimization

## Project Overview

This project simulates a Business Analyst engagement for a mortgage lender experiencing increased loan processing times and declining approval rates.

The objective is to identify bottlenecks in the mortgage application funnel, analyze operational performance, and recommend process improvements.

---

## Business Problem

Mortgage approval turnaround time increased from 8 days to 17 days.

Management needs to identify:

- Process bottlenecks
- Drop-off stages
- SLA breaches
- Operational inefficiencies

---

## Objectives

- Analyze the mortgage application funnel
- Identify bottlenecks
- Improve approval turnaround time
- Increase operational efficiency
